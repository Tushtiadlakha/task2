### Requirements

- Node

#### Dependencies

- axios
- bootstrap
- react
- react-dom
- react-scripts
- react-toastify
- reactstrap

## Building steps

1. Run `npm install`
   This will install dependencies.

2. Run `npm start`
   React application will start on port `3000` of the `localhost`.

3. Run `npm run build`
   This will build the artifacts in `/build` folder.
